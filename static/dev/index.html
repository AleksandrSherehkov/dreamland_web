Документация для сторителей зон и скриптописателей

Строительтсво

  • Справочник по редактору зон OLC: это просто онлайн-копия поваренной книги, доступной также из мира. Справочник на сегодняшний день уже не полный и нуждается в доработке.
  • Справочник по текстовому редактору: редакторы комнат, мобов и т.д. используют общий текстовый редактор описаний.

Феня, скриптовый язык

  • Введение в язык Феня с азов: это слегка обработанные логи бесед, которые Филдс вел с Кинд, объясняя что как работает в Фене
  • Краткий рассказ о потоках: тоже лог разговора Филдса с Анцифером
  • Список триггеров для предметов: все обработчики событий на предметах, которые можно перекрывать из скриптов
  • Список триггеров для персонажей: все обработчики событий на персонажах и мобах, которые можно перекрывать из скриптов
  • Формальное описание грамматики языка: сюда стоит заглянуть за строгим описанием всех возможностей синтаксиса
