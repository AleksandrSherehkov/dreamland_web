Редактирование феневых скриптов


Синтаксис команды codesource (cs)

Вот что выводится в мире, если набрать cs:

Чтение:
     list         - показать список всех существующих codesources
     read <номер> - прочитать cs по номеру из списка
     copy <номер> - скопировать cs из списка в буфер редактора

Редактирование:
     subj <текст> - указать тему cs; удобно тут же указывать версию
     bin <id>     - прочесть текст cs из https://pastebin.com/<id>
     paste        - вставить текст cs из буфера редактора
     + <строка>   - добавить одну строку в текст cs
     -            - удалить последнюю строку из текста cs
     show         - показать редактируемый cs
     clear        - очистить редактируемый cs

Выполнение:
     post         - выполнить cs, далее можно делать eval blablaInit() для объявленной там функции


Использование pastebin

Можно выкладывать длинные скрипты на https://pastebin.com, а затем указать их идентификатор команде 'cs bin'.

Пример создания нового скрипта:

cs subj my cool script, v1.0
cs bin IUEoie83e
cs post
eval ...



Встроенный редактор в веб-клиенте

Если вы используете веб-клиент, можно редактировать буфер прямо из браузера.
Для этого делайте так:

Пример создания нового скрипта:
cs subj my cool script, v1.0
webedit -- открывает окно, в которое можно вбить или вставить текст, затем нажать Save.
cs paste
cs post
eval ...

Пример редактирования существующего:
cs list
cs copy 33
webedit -- текст скрипта с номером 33 окажется в окне, там его можно видоизменить и нажать Save
cs subj my cool script, v1.1
cs paste
cs post
eval ...
